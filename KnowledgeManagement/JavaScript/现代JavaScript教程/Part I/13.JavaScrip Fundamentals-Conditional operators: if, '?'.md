# Conditional operators: if, '?'

## The "if" statement

```js
if (boolean expression) statement;

if (boolean expression) {
  statement1;
  statement2;
}
```

## Boolean conversion

-  __falsy__ - `0`, `""`, `null`, `undefined`, `NaN`.
- __truthy__ - except above.

## The "else" clause

```js
if (boolean expression) {

} else {

}
```

## "else if"

```js
if (expression) {
  
} else if (expression) {

} else {

}
```

## Ternary operator '?'

```js
result = condition ? value1 : value2;
```
